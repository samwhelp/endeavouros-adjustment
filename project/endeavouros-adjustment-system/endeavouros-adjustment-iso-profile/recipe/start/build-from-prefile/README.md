## Howto

### prepare

to install [archiso](https://archlinux.org/packages/extra/any/archiso/)

``` sh
make prepare
```


### build

to build iso

``` sh
make build
```


### run_iso

to test iso

```
make run_iso
```

### clean

to clean dir: [work, out]

``` sh
make clean
```
